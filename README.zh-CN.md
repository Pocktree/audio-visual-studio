# Audio Visual Studio

[English](README.md) | [简体中文](README.zh-CN.md)

---

实时视觉特效测试平台，采用 React + Framer Motion 构建。用于显示测试、运动模糊分析、排版动画和音频可视化。

---

## 设计概述

这是一个**模块化视觉测试框架**，通过不同的视觉特效模块进行循环测试：

- **GRID** — 显示测试（色彩、均匀性、BFI、白平衡）
- **BIG** — 排版与波浪动画
- **SCROLL** — 运动模糊与滚动流畅度测试
- **RAIN** — 粒子模拟
- **AUDIO-VISUAL** — 音频响应可视化

所有模块均可通过键盘快捷键或屏幕控制面板进行操作。

---

## 全局控制

### 快捷键


| 按键   | 功能             |
| ---- | -------------- |
| `P`  | 切换自动播放（循环切换模块） |
| `空格` | 切换到下一个模块       |
| `F`  | 切换全屏           |


### 导航栏

鼠标靠近底部边缘时显示在底部，显示当前模块名称和 auto/paused 状态。

## 模块

### 1. GRID

用于**显示测试**的动态彩色网格。

#### 功能


| 功能           | 描述                       |
| ------------ | ------------------------ |
| 自动轮播         | 每5秒随机改变网格大小（1×1 到 N×N）   |
| 色彩空间         | 切换 Display-P3 和 sRGB 色域  |
| BFI 测试       | 黑帧插入，用于运动模糊 reduction 测试 |
| COLOR OFFSET | 白平衡校准，支持 R/G/B 偏移调整      |
| 均匀性测试        | 7 级灰度图案，用于面板均匀性验证        |
| 灰阶深度测试       | 渐变可视化，用于位深度/抖动分析         |


#### 快捷键


| 按键        | 功能                |
| --------- | ----------------- |
| `[` / `]` | 切换校准档位（低/中/高）     |
| `↑` / `↓` | 调整 R/G/B 偏移值 (±1) |
| `1` - `7` | 触发均匀性测试图案         |
| `D`       | 切换灰阶深度测试模式        |


---

### 2. BIG

全窗口**动态排版字体**波浪效果。

#### 功能


| 功能    | 描述             |
| ----- | -------------- |
| 随机字母  | 显示随机大写字母并旋转    |
| 波浪效果  | 正弦波浪动画贯穿视窗     |
| 倒置模式  | 反转颜色           |
| 轮廓模式  | 文字描边效果         |
| 字体粗细  | 可调整字重（100-900） |
| 超扫描标尺 | 几何畸变与边框校准标尺    |


#### 快捷键


| 按键  | 功能     |
| --- | ------ |
| `7` | 减小波浪振幅 |
| `8` | 增大波浪振幅 |
| `9` | 减小波浪频率 |
| `0` | 增大波浪频率 |


---

### 3. SCROLL

用于运动模糊测试的**平滑滚动文字**动画。

#### 功能


| 功能   | 描述                       |
| ---- | ------------------------ |
| 滚动动画 | 持续水平滚动文字                 |
| 倒置模式 | 反转颜色                     |
| 颜色模式 | 切换黑白/随机颜色                |
| 波浪边缘 | 切换波浪边缘效果                 |
| 追踪模式 | 模拟摄像机追踪，用于 GTG（灰阶响应时间）测试 |


#### 快捷键（当 追踪模式 激活时）


| 按键        | 功能                      |
| --------- | ----------------------- |
| `[` / `]` | 调整滚动速度（480/960/1440 Hz） |
| `+` / `-` | 调整追踪缩放                  |
| `←` / `→` | 调整虚拟阴影长度                |


---

### 4. RAIN

**雨滴粒子模拟**效果。

#### 功能


| 功能   | 描述        |
| ---- | --------- |
| 粒子系统 | 基于物理的雨滴模拟 |
| 速度控制 | 调整雨滴下落速度  |
| 密度控制 | 调整雨滴数量    |
| 网格叠加 | 显示参考网格    |
| 标签   | 显示速度/密度标签 |
| 浅色模式 | 切换浅色背景    |
| 配色方案 | 默认或水蓝色主题  |


#### 快捷键

| 按键 | 功能 |
|------|------|
| `+` | 密度增加500（压力测试） |
| `-` | 密度减少500（最少10） |

---

### 5. AUDIO-VISUAL

**音频响应式排版** + 流体感 Shader 可视化。

#### 功能


| 功能           | 描述                    |
| ------------ | --------------------- |
| Fluid Shader | 实时流体感模拟可视化            |
| 音频响应         | 排版随音频输入响应             |
| 色彩测试         | SMPTE 彩条、灰度或 fluid 模式 |
| 字体粗细         | 可调整字重                 |
| 间距           | 可调整字间距                |
| 颜色反转         | 切换颜色反转                |
| 故障效果         | 随机故障/抖动效果             |
| 实时计时器        | 显示运行时间                |


#### 快捷键

*无*
